# TODO Planner — Product Specification

## Overview

A simple, lightweight web-based TODO planner that allows users to manage their daily tasks efficiently. The application focuses on simplicity, usability, and a clean user interface.

---

## Goals

- Allow users to create, read, update, and delete tasks (CRUD).
- Provide a clean, responsive UI accessible on both desktop and mobile.

---

## Tech Stack

| Layer      | Technology          |
|------------|---------------------|
| Frontend   | HTML, CSS, JavaScript (Vanilla) |
| Storage    | mysql  |
| Backend    | Java springboot |

---

## Features

 1. Add a Task
 2. View Tasks
 3. Complete a Task
 4. Delete a Task
 5. Edit a Task
 6. Filter Tasks
 7. Bulk Actions
 8. Persist Data
